## 📖 Project Overview

The **Employee Management System** is a menu-driven Python application that utilizes various Object-Oriented Programming (OOP) concepts. The system models employee data and operations related to adding and viewing structural records while tracking specific corporate roles.

This project is designed for students learning **Object-Oriented Programming Fundamentals**.

---

# ✨ Features

## 👤 1. Create a Person
- Instantiate standard person entities
- Sets base name and age characteristics

---

## 👔 2. Create an Employee
- Inherits attributes from the Person class
- Manages encapsulated system keys such as ID and salary parameters

---

## 💼 3. Create a Manager
- Extends the core employee structure into managerial tracks
- Assigns specific operating corporate departments

---

## 🔍 4. Show Details
- Displays filtered dynamic structures:
  - Base Person Profiles
  - Standard Employee Logs
  - Executive Manager Subclasses

---

## 🚪 5. Exit Program
- Gracefully clears run space memory
- Automatically triggers native instance destructors
